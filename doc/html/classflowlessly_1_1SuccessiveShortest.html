void flowlessly::SuccessiveShortest::successiveShortestPath ( )

Implements the successive shortest path algorithm without potentials. As a result, it uses Bellman-Ford to compute the paths. NOTE: It changes the graph.

void flowlessly::SuccessiveShortest::successiveShortestPathPotentials ( )

Implements the successive shortest path algorithm with potentials. It uses Dijkstra's algorithm with heaps to compute the paths. NOTE: It changes the graph.
